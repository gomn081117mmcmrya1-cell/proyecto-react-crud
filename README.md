# 🚀 proyecto-react-crud

## 📌 Descripción del Proyecto
Aplicación web Full Stack integrada por un backend en Django y un frontend en React. El sistema permite realizar operaciones CRUD administrando los datos a través de una base de datos relacional SQLite.

## 🛠️ Tecnologías Utilizadas
* **Frontend:** React.js, JavaScript, HTML5, CSS3.
* **Backend:** Python, Django, Django REST Framework.
* **Base de Datos:** SQLite3.
* **Control de Versiones:** Git, GitHub.

## 📥 Instrucciones de Instalación
1. Clonar el repositorio: `git clone https://github.com/gomn081117mmcmrya1-cell/proyecto-react-crud.git`
2. Backend: Ejecutar `python manage.py runserver`
3. Frontend: En la carpeta `mi-app-react`, ejecutar `npm install` y `npm start`
